# 🎮《存在四部曲》——哲学向2D平台跳跃游戏

## 📖 项目介绍  

一个以存在主义哲学为核心的独立游戏，通过**4个主题关卡**探讨人生本质问题：  
- 每关采用独特玩法机制  
- 融合中国式存在焦虑的叙事表达  
- 开发语言：HTML5/CSS3/JavaScript  

---

## 🎯 目标与意义  

- 完整实践2D闯关游戏开发全流程
- 掌握游戏循环、物理引擎、动画系统等核心模块
- 构建可扩展的代码架构



---

## 🚀 安装与运行  

- **下载并解压** 游戏文件。  

- **使用浏览器打开** `index.html` 文件，即可游玩。  

- **游戏机制**：闯关成功后，游戏会自动跳转至下一关。
